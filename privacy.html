Privacy Policy — EnDroit

Last updated: May 2026

1. Introduction

This Privacy Policy describes how the EnDroit application, developed and operated by Gillian Masse, handles information in connection with its use of the TikTok Content Posting API and other third-party services. The Application is a private, non-commercial desktop tool used exclusively by its developer.

2. Data We Do Not Collect

The Application does not collect, process, or store any personal data from TikTok users or any other individuals:

  • No TikTok user profile data is accessed or stored
  • No TikTok follower, comment, or engagement data is read
  • No analytics or behavioral data is collected
  • No cookies or tracking technologies are used

3. TikTok API Data Usage

The Application uses the following TikTok API scopes:

  • video.upload — to upload a locally rendered MP4 file to TikTok's servers
  • video.publish — to publish the uploaded video to the Developer's TikTok profile
  • user.info.basic — to identify the authenticated TikTok account

No TikTok data is read, extracted, or stored. API tokens are stored in a local environment file on the Developer's machine and are never shared or transmitted to third parties.

4. Third-Party Services

  • Anthropic Claude API — receives video topic text; returns generated scripts
  • TikTok Content Posting API — receives video files and metadata for publication
  • Pexels API — returns stock video footage based on keyword queries

5. Data Retention

Video files and scripts are retained locally on the Developer's machine. No cloud storage of personal data is performed.
