Privacy Policy

Last updated: May 2026 · EnDroit by Gillian Masse

EnDroit is a private, single-user desktop application. It does not collect, store, or share any personal data from TikTok users or any other individuals.

1. Introduction

This Privacy Policy describes how the EnDroit application ("the Application"), developed and operated by Gillian Masse ("the Developer"), handles information in connection with its use of the TikTok Content Posting API and other third-party services.

The Application is a private, non-commercial desktop tool used exclusively by its developer. It has no public users, no registration system, and no user-facing interface.

2. Data We Do Not Collect

The Application does not collect, process, or store any personal data from TikTok users or any other individuals. Specifically:

  • No TikTok user profile data is accessed or stored
  • No TikTok follower, comment, or engagement data is read
  • No analytics or behavioral data is collected
  • No cookies or tracking technologies are used
  • No third-party user data of any kind is processed

3. Data Used by the Application

The Application processes only the following data, all of which belongs exclusively to the Developer:

  • Video content (MP4 files) generated locally by the Developer's pipeline
  • Video metadata (title, description, hashtags) generated from the video script
  • TikTok access tokens used to authenticate API requests on behalf of the Developer's own TikTok account

This data is stored locally on the Developer's machine and is never transmitted to any party other than the relevant API provider for the sole purpose of publishing video content.

4. TikTok API Data Usage

The Application uses the following TikTok API scopes:

  • user.info.basic — to identify the authenticated TikTok account (open ID, display name)
  • video.upload — to upload a locally rendered MP4 file to TikTok's servers
  • video.publish — to publish the uploaded video to the Developer's TikTok profile

No TikTok data is read, extracted, or stored by the Application beyond what is strictly necessary to complete a video upload. API tokens are stored in a local environment file (.env) on the Developer's machine and are never shared, logged externally, or transmitted to third parties.

5. Third-Party Services

The Application interacts with the following third-party services for content creation purposes:

  • Anthropic Claude API — receives video topic text; returns generated scripts
  • TikTok Content Posting API — receives video files and metadata for publication
  • Pexels API — returns stock video footage based on keyword queries
  • ElevenLabs API — receives script text; returns synthesized voice audio

None of these services receive personal data of any third party. All interactions are strictly between the Application and the respective API, on behalf of the Developer.

6. Data Retention

The Application retains video files, scripts, and audio files locally on the Developer's machine for archival purposes. No cloud storage of personal data is performed. API tokens are stored in a local .env file and can be revoked at any time via the respective developer portals.

7. Your Rights (GDPR)

As the Application does not process any personal data belonging to third parties, no data subject rights (access, rectification, deletion, portability) are applicable in the context of this Application. If you believe your data has been processed in error, please contact the Developer directly.

8. Security

API credentials and access tokens are stored locally in an environment file (.env) on the Developer's personal machine. The Application does not transmit credentials to any external logging or monitoring service.

9. Changes to This Policy

This Privacy Policy may be updated from time to time. Changes will be reflected by an updated "Last updated" date at the top of this page.

10. Contact
