Features

Workflow

How it works

1

Choose a topic

The creator selects a topic to cover — currently legal education (e.g. tenant rights, employment law, consumer protection). The application generates a complete video script using Claude AI.

2

Generate & preview the video

The script is rendered into a 60–70 second vertical video using Remotion. The creator reviews the result locally in a preview studio before publishing.

3

Publish to TikTok

The rendered MP4 is uploaded to TikTok's servers via the Content Posting API (video.upload) and published to the creator's profile (video.publish).

About EnDroit

EnDroit is a personal project by Gillian Masse, a law and computer science student in France. The application is a private desktop video production pipeline — not a public service or commercial product.

It currently produces short-form educational videos on legal topics, but is built as a general-purpose video production tool. It runs entirely on the creator's local machine. All content is published to the developer's own TikTok account.

TikTok API Usage

EnDroit uses the TikTok Content Posting API exclusively to upload and publish video content created by the developer:

  • video.upload — transfers the rendered MP4 to TikTok's servers
  • video.publish — publishes the video to the creator's own profile

No TikTok user data is accessed, read, or stored. Contact: gillian83.masse@gmail.com
